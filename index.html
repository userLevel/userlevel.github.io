userLevel sequoia

introduction

userLevel «sequoia» is a modern, minimalist outline processor (otherwise known as an “outliner”) which offers an ideal, orthogonal set of outlining features drawn from the NeXTSTEP outliner NoteBook, certain modern outliners, namely OmniOutliner, Outlinely FoldingText and OneNote, the modern internet–based Workflowy, DynaList and Little Outliner, “classic” outliners (including those for the Apple II, MS–DOS and Macintosh) such as ThinkTank, GrandView, MORE and Acta and finally examples from nontraditional outliners, including text editors which offer “folding,” including BBEdit, Sublime Text, Visual Studio Code and Atom as well as Smartsheet, Lighten, MindNode and Tree (a horizontal outliner).
